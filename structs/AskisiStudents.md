**Άσκηση 1)**
Δημιούργησε μια δομή (structure) με όνομα Student, η οποία να περιλαμβάνει τα μέλη:
name (όνομα)
age (ηλικία)
total_marks (συνολική βαθμολογία)
Να γραφτεί ένα πρόγραμμα σε C το οποίο:  
να εισάγει (input) δεδομένα για δύο φοιτητές,  
να εμφανίζει τις πληροφορίες τους,
και να υπολογίζει τον μέσο όρο των συνολικών βαθμολογιών.

**Άσκηση 2)**
Δημιούργησε μια δομή (structure) με όνομα Book, για την αποθήκευση στοιχείων βιβλίου:
title (τίτλος)
author (συγγραφέας)
price (τιμή)
Να γραφτεί ένα πρόγραμμα σε C το οποίο: 
να εισάγει στοιχεία για τρία βιβλία,
να εντοπίζει το πιο ακριβό και το φθηνότερο βιβλίο,
και να εμφανίζει τις πληροφορίες τους.
